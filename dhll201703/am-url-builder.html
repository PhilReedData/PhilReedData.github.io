Uses API v1.0. Works best in Google Chrome, so that the JSON data returned will be displayed nicely.

Access to the Adam Matthew API is restricted by API key and by IP range. Terms and conditions apply.

Mass Observation: Documents

Documents represent logical whole entities of pages or images such as books, manuscripts or magazines. Full text data is only returned when requesting a specific single document by its 'id' and never when requesting multiple documents by searching to increase performance.

Mass Observation: Sections

Sections represent the subdivisions of a document. Conceptually within the collections these can be entities such as sections, chapters, articles and issues. Full text data is not returned when requesting sections.
